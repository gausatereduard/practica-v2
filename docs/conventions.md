# Naming Conventions

## Components

kebab-case

Example:

user-card.tsx

---

## Hooks

camelCase with use prefix

Example:

useUser.ts

---

## Stores

useStoreName.ts

Example:

useEditorStore.ts

---

## Utilities

camelCase

Example:

formatDate.ts

---

## Types

PascalCase

Example:

User
EditorState

---

## Constants

UPPER_SNAKE_CASE
