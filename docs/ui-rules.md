# UI Rules

Default stack:

- shadcn/ui
- Tailwind

---

## Layout

Use:

- max-width containers
- responsive spacing
- grid when appropriate

---

## Forms

Use:

- react-hook-form
- zod

---

## Accessibility

Required:

- keyboard navigation
- visible focus states
- semantic HTML

---

## Dark Mode

Every component must support:

- light mode
- dark mode

without custom overrides whenever possible.
